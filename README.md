# ecmwf-opendata

A package to download ECMWF open data

```python    

fromecmwf.opendataimportClient

client = Client("https://....")

client.retrieve(
   date=-1,
   time=6,
   step=144,
   stream="waef",
   type="ef",
   target="data.grib",
   param='mwd',
)

```
